# GEO GUIDE - AI-Powered Mineral Prediction System

Machine learning system for gold prospectivity mapping using remote sensing data.

## Features
- Unsupervised K-Means clustering
- PCA dimensionality reduction
- Real-time training on uploaded data
- Comprehensive visualizations

## Usage
1. Upload stacked_features.tif (multi-band raster)
2. Upload gold_occurences.gpkg (gold locations)
3. Click "Run Analysis"
4. View results and download reports
